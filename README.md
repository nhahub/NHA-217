                                              Nova Shop  
 Overview
Nova Shop is a modern e-commerce web application built using React and Tailwind CSS. The application provides a smooth shopping experience for users and a powerful management interface for admins.
It supports essential e-commerce functionalities such as product browsing, searching, filtering, user authentication, and admin management.
________________________________________
 Features
 User Features
•	Browse products
•	Search products by name
•	Filter products by category, price, or other attributes
•	View product details
•	Add products to cart
•	User authentication (login / register)
 Admin Features
•	Admin dashboard
•	Add, edit, and delete products
•	Manage categories
•	View users
•	Control product availability
________________________________________
Technologies Used
•	Frontend: React
•	Styling: Tailwind CSS
•	Routing: React Router
•	Tools: Git, GitHub, VS Code
