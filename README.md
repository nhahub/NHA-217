# Nova Shop

## Overview
Nova Shop is an e-commerce application built using React and Tailwind CSS.
It provides product browsing, searching, filtering, and role-based access for users and administrators.

## Features

### User Features
- Browse products
- Search products
- Filter products by category and price
- View product details
- Add products to cart
- User authentication (login and register)

### Admin Features
- Admin dashboard
- Add, edit, and delete products
- Manage categories
- View users
- Control product availability

## Technologies Used
- React
- Tailwind CSS
- React Router
- Git and GitHub
- VS Code
